Database and data model(s)
==========================
The observational data and metadata schema used in the OpenCDMS database imports and extents 3 external data models:

1. The WIGOS metadata standard (station and sensor metadata).
2. The WMO Core Metadata Profile (v2) (discovery metadata).
3. The WMO Climate Data Model (observation, processing and provenance data and metadata).
